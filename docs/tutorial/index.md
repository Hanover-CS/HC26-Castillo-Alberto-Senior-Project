# Tutorial: Using Scenes & Signals to Build Brick Breaker in Godot
---

This tutorial teaches you the essential Godot 4.0 skills needed to build the core systems behind a *Brick Breaker–style* game. Using simple and readable GDScript, you'll learn how to structure scenes, implement a paddle and ball, detect collisions, and connect gameplay systems using Godot signals.

> **Target audience:** College students with programming knowledge.  
>
> **Learning objectives:**
> - Understand how Godot scenes and nodes work together.
> - Create reusable scenes for the paddle, ball, and bricks.
> - Use Godot signals to communicate between gameplay objects.
> - Build the essential pieces of a simple Brick Breaker prototype.
