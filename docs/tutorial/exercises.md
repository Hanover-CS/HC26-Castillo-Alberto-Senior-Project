# Exercises
---

Try these challenges to extend your prototype and deepen your understanding.

## 1. Add a Life System
---


- Detect when the ball falls off-screen  
- Emit a `life_lost` signal  
- Reset the ball  
- Track and display remaining lives  

---

## 2. Add a Power-Up System

- Create a PowerUp scene  
- Spawn it when a brick is destroyed  
- Make it fall  
- Detect collision with the paddle  
- Trigger an effect (bigger paddle, speed boost, etc.)
