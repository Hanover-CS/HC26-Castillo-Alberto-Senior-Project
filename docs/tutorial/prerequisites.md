# Prerequisites

### Tools
- **Godot 4.x** (Stable release)
- A computer that can run Godot
- A GitHub repo with GitHub Pages enabled

### Knowledge
- Basic programming (variables, if/else, functions)
- Very basic git familiarity (to commit and push your files)
