# Project Setup

## 1) Create a new Godot project
1. Open Godot → **New Project** → choose a folder → name it (e.g., `PaddleBall`)
2. Create a main scene: **Scene → New Scene → Node2D** as root → save as `Main.tscn`.

## 2) Input Map
- **Project → Project Settings → Input Map**
- Add actions: `ui_left`, `ui_right`
- Bind to Left/Right arrow keys
